# Track: GUI UX Enhancements

## Overview
Address user feedback regarding GUI responsiveness and transparency:
1. **Threading**: Run scrapers in background threads to prevent UI freezing.
2. **Manual Controls**: Add a button to manually trigger the selected scraper strategy.
3. **Progress Indicators**: Add an indeterminate progress bar for long-running tasks.
4. **Transparency**: Log LLM prompts and payloads to the console for debugging.
