# Specification: Basic Archive Scanning and Metadata Extraction\n\n## Overview\nThis track focuses on the foundational ability to discover comic archives within a file system and extract metadata from their filenames.\n\n## Functional Requirements\n- **Directory Scanning**: Recursively scan a provided directory for supported comic archive formats (\`.cbz\`, \`.cbr\`, \`.cb7\`).\n- **Metadata Extraction**: Parse archive filenames to extract:\n    - **Series**: The name of the comic series.\n    - **Volume**: The volume number (if present).\n    - **Issue/Number**: The issue or chapter number.\n    - **Year**: The publication year (if present in the filename).\n- **Data Model**: Populate a \`ComicInfo\` object with the extracted metadata.\n\n## Technical Details\n- **Libraries**: Use \`os\` and \`pathlib\` for file system operations.\n- **Regex**: Refine existing \`FilenameScraper\` regex to handle common patterns like:\n    - \`Series V01 #01 (2024).cbz\`\n    - \`Series 001.cbr\`\n- **Output**: The extracted metadata should be ready to be passed to the XML generation logic.
