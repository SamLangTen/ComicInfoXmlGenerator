# Track: GUI Advanced Features

## Overview
Advance the GUI functionality with power-user features:
1. **Selection Visibility**: Improve the visual state of selected items in the file list.
2. **Multi-selection**: Allow selecting multiple archives to process in batch.
3. **Application Modes**:
   - **Overwrite**: Replace current metadata fields with scraper results.
   - **Fill Gaps**: Only fill empty fields, preserving existing metadata.
